# Vending Machine State Pattern Example

## Описание
Реализует автомат по продаже напитков с различными состояниями и случайным бонусом (двойная выдача).

## Состояния автомата
- **NoCoinState** — ожидание монеты
- **HasCoinState** — монета вставлена, ожидание выдачи
- **SoldOutState** — напитки закончились
- **WinnerState** — состояние победителя (бонусная выдача)

## Как работает
1. Автомат начинает в состоянии `NoCoinState`
2. При вставке монеты переходит в `WinnerState` (имитация лотереи)
3. В состоянии `WinnerState` есть шанс получить дополнительный напиток
4. Возможен возврат монеты с разными сценариями

## Логика переходов
- Вставка монеты → переход в `WinnerState`
- Выдача напитка → переход в `NoCoinState` (или `SoldOutState` если напитки кончились)
- Возврат монеты → возможен переход в `NoCoinState` или остаться в текущем состоянии

## Пример вывода
```
Вставьте монету сначала!
Монета вставлена.
Монетка вернулась!
Напитки закончились!
```
