# Adapter Pattern Example

## Описание
Позволяет использовать старые устройства с LPT-интерфейсом в новой системе, которая ожидает USB.

## Компоненты
- **USBPrinter, USBScanner** — целевые интерфейсы (новые стандарты)
- **OldLPTPrinter, OldLPTScanner** — старые классы с несовместимым интерфейсом
- **LPTToUSBAdapter, LPTToScanerAdapter** — адаптеры для конкретных устройств
- **UniversalAdapter** — универсальный адаптер для работы с обоими типами

## Как работает
1. Создаются старые устройства (принтер, сканер)
2. Адаптеры оборачивают старые устройства и представляют USB-интерфейс

## Почему Adapter лучше изменения старого класса
- **Старый класс не модифицируется** — нет риска сломать работающий код
- **Обратная совместимость** — существующий код продолжает работать
- **Разделение ответственности** — адаптер отвечает только за преобразование интерфейсов

## Пример вывода
```
Старый принтер печатает по LPT: Документ (адаптировано для USB) quality: 1
Старый сканер печатает по LPT: Документы (адаптировано для USB) quality: 2
Старый принтер печатает по LPT: Документы (адаптировано для USB) quality: 3
```
